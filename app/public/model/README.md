# Model Files

This folder should contain your TFJS model files after training.

## How to get these files

1. Run the Colab notebooks in `ml/` (in order: 01 → 02 → 03 → 04)
2. Download `tfjs_model_for_app.zip` from notebook 03
3. Unzip and place these files here:

```
model/
  model.json          ← TFJS model graph
  group1-*.bin        ← Model weights (may be multiple shards)
  labels.json         ← Class index → class key mapping (REQUIRED)
```

## Why not committed?

Model binary files (`.bin`) are too large for GitHub (they can be 3–10 MB each).
The full trained model is available on HuggingFace Hub (link in README) or
generated by running the Colab training pipeline.
